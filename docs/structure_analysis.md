# 機械手臂結構分析與評估

## 一、基本規格

| 項目 | 數值 |
|------|------|
| 節數 | 2 段 |
| 每段長度 | 30 cm |
| 總展開長度 | 60 cm |
| 末端預估負載 | 2 kg |
| 控制方式 | RS485 Modbus-RTU 步進電機 |

---

## 二、力矩計算

### 單段伸展（肘部）
```
τ = 2 kg × 9.81 m/s² × 0.30 m ≈ 5.9 N·m
```

### 兩段完全伸直（肩部最大力矩）
```
τ = 2 kg × 9.81 m/s² × 0.60 m ≈ 11.8 N·m
（加上手臂自重 + 動態加速 → 實際需求 20~30 N·m）
```

### 結論
| 軸 | 建議力矩 |
|----|----------|
| 肩軸（J1） | 20~30 N·m（關鍵） |
| 肘軸（J2） | 5~10 N·m |

> ⚠️ 肩軸扭矩是穩定與否的分水嶺，馬達選型不足會導致丟步或無法正常運作

---

## 三、控制架構建議（三層分離）

```
┌─────────────────────────────────┐
│  UI 層（Flask / Streamlit）      │  使用者操作介面
│  → 送指令給 Motion Engine        │
└─────────────────┬───────────────┘
                  │
┌─────────────────▼───────────────┐
│  Motion Engine（軌跡規劃）       │  速度曲線、S-curve、插補
│  → 計算每軸脈衝序列              │
└─────────────────┬───────────────┘
                  │
┌─────────────────▼───────────────┐
│  RS485 驅動層（寄存器讀寫）      │  只負責 Modbus 通訊
│  → 直接操作每台步進控制器        │
└─────────────────────────────────┘
```

---

## 四、安全設計要點

1. **限位開關**：每軸至少一組硬體限位（正 + 負方向）
2. **軟限位**：透過 0x006E~0x0071 設定脈衝上下限
3. **急停功能**：0x00C8 = 256（廣播到所有軸同時急停）
4. **歸零（Home）**：每次開機必須執行歸零序列

## 六、與現有系統整合

- **通訊界面**：RS485（同 pi52 煮麵系統架構）
- **建議控制主機**：pi53（獨立於 pi52 煮麵系統）
- **程式語言**：Python（`minimalmodbus` 或 `pymodbus`）
- **Web UI**：可整合到現有 Streamlit Dashboard 或獨立 Flask 服務
